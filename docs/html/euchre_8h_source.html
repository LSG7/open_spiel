18 // Full implementation of the classic trick taking game Euchre.
19 //
20 // https://en.wikipedia.org/wiki/Euchre
21 // https://www.pagat.com/euchre/euchre.html
22 //
23 // This implementation uses standard North American rules with "super-Euchres",
24 // i.e. the makers lose 4 points if they fail to win a single trick. By default,
25 // only the declarer has the option of playing alone, but optionally the
26 // defenders can go alone as well. The popular variation "stick the dealer" is
27 // enabled by default as it has interesting strategic implications and increases
28 // playability by avoiding drawn hands.

252  return 1 + // Dealer selection
253  (kNumPlayers * kNumTricks) + // Deal hands
