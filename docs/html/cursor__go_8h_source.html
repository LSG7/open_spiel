30 // Game of Go, with a cursor interface. Actions are to move the cursor up, down,
31 // left, or right. Or to pass or place a stone in the current cursor position.
32 // https://en.wikipedia.org/wiki/Go_(game)
33 //
34 // Parameters:
35 // "komi" float (default 7.5) compensation for white
36 // "board_size" int (default 19) rows of the board
37 // "handicap" int (default 0) number of handicap stones for black
38 // "max_cursor_moves" int (default 100) maximum number of cursor moves
39 // before a player must pass or play.
40 //
41 // Handicap stones assume a 19x19 board.

72 // In theory Go games have no length limit, but we limit them to twice the
73 // number of points on the board for practicality - only random games last
74 // this long.
75 // The maximum number of cursor go moves is greater by a factor of
76 // (1+max_cursor_moves).
77 inline int MaxGameLength(int board_size) { return board_size * board_size * 2; }
