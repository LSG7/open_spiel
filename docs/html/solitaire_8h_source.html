28 // An implementation of klondike solitaire:
29 // https://en.wikipedia.org/wiki/Klondike_(solitaire) More specifically, it is
30 // K+ solitaire, which allows the player to play any card from the deck/waste
31 // that would normally become playable after some number of draws in standard
32 // klondike solitaire. For a more in-depth description of K+ solitaire, see
33 // http://web.engr.oregonstate.edu/~afern/papers/solitaire.pdf. This
34 // implementation also gives rewards at intermediate states like most electronic
35 // versions of solitaire do, rather than only at terminal states.
