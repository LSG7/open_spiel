29 // The AFCCE auxiliary game is very similar to the EFCCE auxiliary game. The
30 // only difference is that the number of deviations is limited to 1. Once the
31 // player has deviated, they can no longer deviate and must follow for the
32 // rest of the game.

94  game_->NewInitialState(), config_, mu_,
95  FollowAction(), DefectAction());

105 
106  private:
