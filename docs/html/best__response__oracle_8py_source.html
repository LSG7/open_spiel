98 
99  Args:
100  game: The game on which the optimization process takes place.
101  training_parameters: List of list of dicts: one list per player, one dict
102  per selected agent in the pool for each player,
103  each dictionary containing the following fields:
104  - policy: the policy from which to start training.
105  - total_policies: A list of all policy.Policy strategies used for
106  training, including the one for the current player. Either
107  marginalized or joint strategies are accepted.
108  - current_player: Integer representing the current player.
109  - probabilities_of_playing_policies: A list of arrays representing, per
110  player, the probabilities of playing each policy in total_policies for
111  the same player.
112  strategy_sampler: Callable that samples strategies from `total_policies`
113  using `probabilities_of_playing_policies`. It only samples one joint
114  "action" for all players. Implemented to be able to take into account
115  joint probabilities of action.
116  using_joint_strategies: Whether the meta-strategies sent are joint (True)
117  or marginalized.
118  **oracle_specific_execution_kwargs: Other set of arguments, for
119  compatibility purposes. Can for example represent whether to Rectify

121 
122  Returns:
123  A list of list of OpenSpiel Policy objects representing the expected
124  best response, following the same structure as training_parameters.
125  """
126  new_policies = []
127  for player_parameters in training_parameters:
128  player_policies = []
129  for params in player_parameters:
130  current_player = params['current_player']
131  total_policies = params['total_policies']
132  probabilities_of_playing_policies = params[
133  'probabilities_of_playing_policies']
134  if using_joint_strategies:
135  aggr_policy = utils.aggregate_joint_policies(
136  game, utils.marginal_to_joint(total_policies),

140  game, total_policies, probabilities_of_playing_policies)

146  # that a best response for current_player can be computed.

148  best_resp = best_response.BestResponsePolicy(game, current_player,
