26// A simple multi-issue bargaining game, based on [1,2]. The rules are based
27// on the description of Section 2.2 of [1]:
28//
29// "Each agent is given a different randomly generated value function, which
30// gives a non-negative value for each item. The value functions are
31// constrained so that: (1) the total value for a user of all items is 10;
32// (2) each item has non-zero value to at least one user; and (3) some items
33// have nonzero value to both users. These constraints enforce that it is not
34// possible for both agents to receive a maximum score, and that no item is
35// worthless to both agents, so the negotiation will be competitive. After 10
36// turns, we allow agents the option to complete the negotiation with no
37// agreement, which is worth 0 points to both users. We use 3 item types
38// (books, hats, balls), and between 5 and 7 total items in the pool."
39//
40// [1] Lewis et al., Deal or no deal? End-to-end learning of negotiation
41// dialogues, 2017. https://arxiv.org/abs/1706.05125
42// [2] David DeVault, Johnathan Mell, and Jonathan Gratch.
43// 2015. Toward Natural Turn-taking in a Virtual Human Negotiation Agent
44//
45// Parameters:
46// "instances_file" string The file containing the boards (default: "")
47// "discount" double Discount factor multiplied each turn after
48// turn 2, applied to (multiplied to reduce) the
49// returns (default = 1.0).
50// "max_turns" integer Maximum total turns before the game ends
51// (default = 10).
52// "prob_end" double Probability of the game ending after each
53// action (only after each player has taken
54// one turn each) (default = 0.0).
