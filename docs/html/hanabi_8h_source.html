18 // Hanabi is a cooperative card game, described here:
19 // https://en.wikipedia.org/wiki/Hanabi_(card_game)
20 //
21 // See https://arxiv.org/abs/1902.00506 for a motivation of Hanabi as an AI
22 // challenge and some initial results. Please cite this paper if you use this
23 // Hanabi wrapper for any research results.
24 //
25 // This implementation is a wrapper for the Hanabi Learning Environment, which
26 // can be found here: https://github.com/deepmind/hanabi-learning-environment
27 //
28 // Since Hanabi relies on an (optional) external dependency, it is not included
29 // in the list of compiled games by default. To enable it, read `install.md`
30 // (TLDR: Set the environment variable OPEN_SPIEL_BUILD_WITH_HANABI to ON).
