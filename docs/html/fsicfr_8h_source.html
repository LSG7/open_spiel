29 // An basic implementation of Neller and Hnath 2011, "Approximating Optimal Dudo
30 // Play with Fixed-Strategy Iteration Counterfactual Regret Minimization"
31 // https://cupola.gettysburg.edu/csfac/2/.
32 //
33 // This implementation currently assumes the following:
34 // - All chance events occur at the start of the game (before any decisions)
35 // - There exists a perfect ranking between a player's chance event outcomes
36 // and their outcome, encoded as a chance_id integer (one per player)
37 //
38 // This implementation was built for and only tested on Liar's dice. For a usage
39 // example, see examples/fsicfr_liars_dice.cc.
40
