26 // Nim:
27 // * Two players take turns removing objects from distinct piles;
28 // * On each turn, a player must remove at least one object,
29 // and may remove any number of objects provided they all come from the
30 // same heap or pile;
31 // * Depending on the version, the goal of the game is either to avoid taking
32 // the last object or to take it. Please see https://en.wikipedia.org/wiki/Nim
33 // for more
