Constructs a list of checkpoint selection tuples to query value function.

Each query tuple (key, query) where key = (pi, pj) and query is
(p1's selected checkpt, ..., p7's selected checkpt) fixes the players in the
game of diplomacy to be played. It may be necessary to play several games with
the same players to form an accurate estimate of the value or payoff for each
player as checkpts contain stochastic policies.

Args:
  base_profile: list of selected checkpts for each player, i.e.,
    a sample from the player strategy profile ([x_i ~ p(x_i)])
  num_checkpts: list of ints, number of strats (or ckpts) per player
Returns:
  Set of query tuples containing a selected checkpoint index for each player.

111 game_results: dictionary of payoffs for each game evaluated, keys are

113 each other and profile indicates pure joint action played by all agents
114 num_checkpts: list of ints, number of strats (or ckpts) per player

116 payoff_matrices: dict of np.arrays (2 x num_checkpts x num_checkpts) with
117 payoffs for two players. keys are pairs above with lowest index agent
118 first
119 """
120 num_players = len(num_checkpts)
121 payoff_matrices = {}
122 for pi, pj in itertools.combinations(range(num_players), 2):
123 key = (pi, pj)
124 payoff_matrices[key] = np.zeros((2, num_checkpts[pi], num_checkpts[pj]))

127 i, j = key
128 ai = profile[i]
129 aj = profile[j]

131 payoff_matrices[key][1, ai, aj] = payoffs[j]
132 return payoff_matrices
