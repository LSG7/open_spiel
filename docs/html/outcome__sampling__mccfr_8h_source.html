27 // An implementation of outcome sampling Monte Carlo Counterfactual Regret
28 // Minimization (CFR). This version is implemented in a way that is closer to
29 // VR-MCCFR, so that it is compatible with the use of baselines to reduce
30 // variance (baseline of 0 is equivalent to the original outcome sampling).
31 //
32 // Lanctot et al. '09: http://mlanctot.info/files/papers/nips09mccfr.pdf
33 // Lanctot, 2013: http://mlanctot.info/files/papers/PhD_Thesis_MarcLanctot.pdf
34 // Schmid et al. '18: https://arxiv.org/abs/1809.03057
35 // Davis, Schmid, & Bowling '19. https://arxiv.org/abs/1907.09633
