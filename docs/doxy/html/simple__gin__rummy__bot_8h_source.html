18// This bot plays about the simplest possible strategy that performs reasonably
19// well and effectively explores the game tree. It's useful both as a test
20// of the game implementation and as a benchmark of playing strength.
21//
22// The strategy can be summarized as follows:
23//
24// If phase == kDraw:
25// Draw the upcard under either of the following two conditions, otherwise
26// draw from the stock:
27// 1) If doing so allows for an immediate knock.
28// 2) If the upcard belongs to a meld, and that meld lowers the deadwood
29// count of the hand. The second part of this condition in relevant in
30// the following example where we would not want to pick up the Js even
31// though it makes three jacks, because it breaks up a better meld
32// thereby increasing the total deadwood count.
33//
34// Upcard: Js
35// +--------------------------+
36// | |
37// |Ac2c3c4c |
38// | 9dTdJdQd |
39// | 3h Jh |
40// +--------------------------+
41//
42// If phase == kDiscard:
43// Always knock if legal, otherwise throw the deadwood card worth the most
44// points, with ties being broken arbitrarily.
45//
46// If phase == kKnock:
47// When laying the hand, the meld arrangement is chosen that minimizes the
48// total deadwood count. If two different meld arrangements are equal in this
49// regard, one is chosen arbitrarily. No layoffs are made if opponent knocks.
