96 // A block of the residual model's network that makes up the 'torso'. It
97 // consists of two convolutional layers (CONV) and two batchnormalization layers
98 // (BN). The activation function is rectified linear unit (RELU). The input to
99 // the layer is added to the output before the final activation function.
100 //
101 // Illustration:
102 // [Input Tensor] --> CONV --> BN --> RELU --> CONV --> BN --> + --> RELU
103 // \___________________________________________________/
104 //
105 // Unlike the input and output blocks, one can specify how many of these torso
106 // blocks they want in their model.

120 // A block of the residual model's network that creates the output. It consists
121 // of a value and policy head. The value head takes the input through one
122 // convoluational layer (CONV), one batch normalization layers (BN), and two
123 // linear layers (LIN). The output activation function is tanh (TANH), the
124 // rectified linear activation function (RELU) is within. The policy head
125 // consists of one convolutional layer, batch normalization layer, and linear
126 // layer. There is no softmax activation function in this layer. The softmax
127 // on the output is applied in the forward function of the residual model.
128 // This design was chosen because the loss function of the residual model
129 // requires the policy logits, not the policy distribution. By providing the
130 // policy logits as output, the residual model can either apply the softmax
131 // activation function, or calculate the loss using Torch's log softmax
132 // function.
133 //
134 // Illustration:
135 // --> CONV --> BN --> RELU --> LIN --> RELU --> LIN --> TANH
136 // [Input Tensor] --
137 // --> CONV --> BN --> RELU --> LIN (no SOFTMAX here)
138 //
139 // There is only one output block per model.
140 class ResOutputBlockImpl : public torch::nn::Module {
141  public:
142  ResOutputBlockImpl(const ResOutputBlockConfig& config);
143  std::vector<torch::Tensor> forward(torch::Tensor x, torch::Tensor mask);
144 
145  private:
146  torch::nn::Conv2d value_conv_;
147  torch::nn::BatchNorm2d value_batch_norm_;
148  torch::nn::Linear value_linear1_;
149  torch::nn::Linear value_linear2_;
150  int value_observation_size_;
151  torch::nn::Conv2d policy_conv_;
152  torch::nn::BatchNorm2d policy_batch_norm_;
153  torch::nn::Linear policy_linear_;
154  int policy_observation_size_;
155 };
