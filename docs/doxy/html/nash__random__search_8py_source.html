31 """Nash Random Search Exploitability Minimizer.
32
33 Implements base functions for regret minimizers to implement.
34
35 Attributes:
36 _game: Pyspiel game.
37 _regret_steps_per_step: Number of regret steps per `step` call (Maximum
38 number in case `stop_early` is true)
39 _rho_tol: If `_compress_nus` is true, minimum probability threshold (
40 Probabilities below `rho_tol` will be filtered out).
41 _compress_nus: Whether to compress nus (Remove nus with low selection
42 probability) or not.
43 _compress_lbd: Penalty term in L1 minimization when compressing nus.
44 _stop_early: Whether to stop regret computation when average regret is lower
45 than `_stop_regret_threshold` or to keep going until
46 `_regret_steps_per_step` steps have been accomplished.
47 _stop_regret_threshold: If `stop_early` is true, average regret threshold
48 under which the algorithm will stop.
49 _policies: List of Policies
50 _value_estimator: Value estimation function.
51 _value_estimation_n: Number of runs to average _value_estimator's result on.
52 """
