24 // Oware (https://en.wikipedia.org/wiki/Oware) is a strategy game within the
25 // family of Mancala games. Several variations of the game exist. This
26 // implementation uses the basic rules as described here:
27 // https://en.wikipedia.org/wiki/Oware or here:
28 // http://www.joansala.com/auale/rules/en/.
29 //
30 // In particular if the opponent has no seeds, the current player must make a
31 // move to give the opponent seeds. If no such move exists the game ends and the
32 // current player collects the seeds in his row. If at the end of an action the
33 // opponent would be left with no seeds because they would all be captured
34 // (a Grand Slam), no seeds are captured instead.
35 //
36 // When the game reaches a state which occurred before, it ends and both players
37 // collect the remaining seeds in their respective rows.
38 //
39 // Note: The Kalah game is also available separately in mancala.{h,cc}.
