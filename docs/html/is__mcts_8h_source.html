65  // Construct an IS-MCTS bot. The parameter max_world_samples controls how many
66  // states are sampled (with replacement!) at the root of the search; use
67  // kUnlimitedWorldStates to have no restriction, and a number larger than
68  // zero to restrict the number). If use_observation_string is true, then
69  // will use ObservationString as a key instead of InformationStateString.
70  // If allow_inconsistent_action_sets is true, then the algorithm handles the
71  // case of differing legal action sets across states with the same state key
72  // (information state string or observation string) which can happen when
73  // using observations or with game that have imperfect recall.

76  // implemented.
