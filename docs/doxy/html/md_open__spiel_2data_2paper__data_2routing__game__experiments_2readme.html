Reproducing routing game experiments

To reproduce the experiments done in [Solving N-player dynamic routing games with congestion: a mean field approach, Cabannes et. al.](https://dl.acm.org/doi/10.5555/3535850.3536033):

  1. If you have not, download python and an IDE to run iPython notebook (either jupyter or VSCode).
  2. Install OpenSpiel using pip install open_spiel or from source.
  3. Download the Experiments.ipynb iPython notebook.
  4. Run the iPython notebook. You might need to download the dependant python libraries.
