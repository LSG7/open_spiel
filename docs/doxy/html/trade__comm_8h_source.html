25// A simple communication game inspired by trading, where agents receive
26// private items, send (arbitrary) utterances, and then have to commit to a
27// trade.
28//
29// First agent receives a random item a set of K unique items. Second agent
30// also receives a random item. Both items are private. Then, the first agent
31// can make a single utterance from a set of K utterances, which the second
32// agent observes. The second agent can do the same (which the first agent
33// observes). Then each of the agents secretly chooses a 1:1 trade action in
34// private. If they choose a compatible trade (i.e. agents trade the item they
35// have for the item the other agent has), they each get a reward of 1.
36// Otherwise, they both get 0.
37//
38// This current variant is the simplest version of more complex communication
39// games for trading. Ultimately, we plan to expand so that the communication is
40// longer and vectorized, and the commitment round is multi-step.
41//
42// Parameters:
43// "num_items" int number of distinct items (K) (default = 10)
44//
