25 // An implementation of the Blotto: https://en.wikipedia.org/wiki/Blotto_game
26 // This version supports n >= 2 players. Each player distributes M coins on N
27 // fields. Each field is won by at most one player: the one with the most
28 // coins on the specific field; if there is a draw, the field is considered
29 // drawn (not won by any player), and hence ignored in the scoring. The winner
30 // is the player with the most won fields: all player have won the same number
31 // of fields, they each receive 0. Otherwise, the winners share 1 / (number of
32 // winners) and losers share -1 / (number of losers), reducing to {-1,0,1} in
33 // the 2-player case.
34 //
35 // Parameters:
36 // "coins" int number of coins each player starts with (default: 10)
37 // "fields" int number of fields (default: 3)
38 // "players" int number of players (default: 2)
