31 class CFRBRSolver(_CFRSolverBase):
32  """Implements the Counterfactual Regret Minimization (CFR-BR) algorithm.
33 
34  This is Counterfactual Regret Minimization against Best Response, from
35  Michael Johanson and al., 2012, Finding Optimal Abstract Strategies in
36  Extensive-Form Games,
37  https://poker.cs.ualberta.ca/publications/AAAI12-cfrbr.pdf).
38 
39  The algorithm
40  computes an approximate Nash policy for n-player zero-sum games, but the
41  implementation is currently restricted to 2-player.
42 
43  It uses an exact Best Response and full tree traversal.
44 
45  One iteration for a n-player game consists of the following:
46 
47  - Compute the BR of each player against the rest of the players.
48  - Then, for each player p sequentially (from player 0 to N-1):
49  - Compute the conterfactual reach probabilities and action values for player
50  p, playing against the set of the BR for all other players.
51  - Update the player `p` policy using these values.
52 
53  CFR-BR should converge with high probability (see the paper), but we can also
54  compute the time-averaged strategy.
55 
56  The implementation reuses the `action_values_vs_best_response` module and
57  thus uses TabularPolicies. This will run only for smallish games.
58  """
