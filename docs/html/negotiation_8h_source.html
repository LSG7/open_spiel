25 // A simple negotiation game where agents propose splits of a group of items,
26 // until the maximum number of turns runs out or an offer is accepted.
27 
28 // This game is inspired by the following papers:
29 // - DeVault et al., Toward natural turn-taking in a virtual human negotiation
30 // agent, 2015.
31 // - Lewis et al., Deal or no deal? End-to-end learning of negotiation
32 // dialogues, 2017.
33 // - Cao et al., Emergent Communication through Negotiation, 2018.
34 // https://arxiv.org/abs/1804.03980
35 //
36 // We use the specific description in Cao et al. 2018. However, we choose
37 // default settings that lead to a smaller game since the values used in the
38 // paper (utterance_dim = 6, num_symbols = 10) could lead to legal action sizes
39 // of 2.17 * 10^8.
40 // TODO(author5): fix this restriction by either (i) adding support for legal
41 // action iterators rather than lists, or (ii) supporting structured actions,
42 // (or both!).
43 //
44 // Parameters:
45 // "enable_proposals" bool open the proposal channel (default = true)
46 // "enable_utterances" bool open the linguistic channel (default = true)
47 // "num_items" int number of distinct items (default = 5)
48 // "num_symbols" int number of distinct symbols (default = 5)
49 // "rng_seed" int seed for the random number generator
50 // (default -1 = not set, seeded by clock)
51 // "utterance_dim" int dimensionality of the utterances, i.e. number
52 // of symbols per utterance (default = 3)
