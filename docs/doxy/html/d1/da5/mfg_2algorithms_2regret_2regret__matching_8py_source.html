25def regret_matching(regrets):
26 regrets = np.array(regrets)
27 regret_plus = regrets * (regrets > 0.0)
28 regrets_sum = np.sum(regret_plus, axis=-1)
29 regret_plus[regrets_sum > 0.0, :] = regret_plus[
30 regrets_sum > 0.0, :
31 ] / regrets_sum[regrets_sum > 0.0].reshape(-1, 1)
32 regret_plus[regrets_sum <= 0.0, :] = (
33 np.ones_like(regret_plus[regrets_sum <= 0.0, :]) / regret_plus.shape[-1]
34 )
35 return regret_plus

42
43 Attributes:

49 _compress_nus: Whether to compress nus (Remove nus with low selection

51 _compress_lbd: Penalty term in L1 minimization when compressing nus.
52 _stop_early: Whether to stop regret computation when average regret is lower
53 than `_stop_regret_threshold` or to keep going until
54 `_regret_steps_per_step` steps have been accomplished.
55 _stop_regret_threshold: If `stop_early` is true, average regret threshold
56 under which the algorithm will stop.
57 _policies: List of Policies
58 _value_estimator: Value estimation function.
59 _value_estimation_n: Number of runs to average _value_estimator's result on.
