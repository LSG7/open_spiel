24 // A simple jeopardy dice game that includes chance nodes.
25 // See http://cs.gettysburg.edu/projects/pig/index.html for details.
26 // Also https://en.wikipedia.org/wiki/Pig_(dice_game)
27 //
28 // Piglet variant: Instead of increasing the running total by the roll results,
29 // it is always increased by a fixed step size of 1 upon rolling anything higher
30 // than a 1. [Note: Internally, this behaviour is modelled with only two chance
31 // outcomes, rolling a 1 or rolling anything higher than that.]
32 // Divide winscore by the average dice outcome != 1 (i.e. by diceoutcomes/2 + 1)
33 // when enabling Piglet to play a game that's roughly equivalent to the
34 // corresponding Pig game. The main advantage of this variant is thus a greatly
35 // reduced state space, making the game accessible to tabular methods.
36 // See also http://cs.gettysburg.edu/~tneller/papers/pig.zip. The original
37 // Piglet variant described there is played with a fair coin and a winscore
38 // of 10. This behaviour can be achieved by setting diceoutcomes = 2, winscore =
39 // 10, piglet = true.
40 //
41 // Parameters:
42 // "diceoutcomes" int number of outcomes of the dice (default = 6)
43 // "horizon" int max number of moves before draw (default = 1000)
44 // "players" int number of players (default = 2)
45 // "winscore" int number of points needed to win (default = 100)
46 // "piglet" bool is piglet variant enabled? (default = false)
