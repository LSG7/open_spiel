30 // A simple simultaneous-move (single- and multi-agent) grid world pathfinding
31 // game.
32 //
33 // Grids can be expressed as ASCII strings, where lower case characters refer
34 // to starting positions, upper case characters refer to destinations,
35 // '.' refers to an empty cell, '*' refers to an wall.
36 //
37 // Parameters:
38 // "grid" int The grid world the agents play in (default below).
39 // "group_reward" double Extra reward (to each agent) if all agents reach
40 // their desitnation (default: 100.0).
41 // "horizon" int Maximum number of steps in an episode (def: 1000).
42 // "players" int Number of players (default: 1, and overridden by
43 // the grid).
44 // "solve_reward" double Reward obtained when reaching the destination
45 // (default: 100.0).
46 // "step_reward" double The reward given to every agent on each per step
47 // (default: -0.01).
48 //
49 // Note: currently, the observations are current non-Markovian because the time
50 // step is not included and the horizon is finite. This can be easily added as
51 // an option if desired.
