40// The underlying C code requires that the number of throws in a game be
41// specified at compile time. Changing it requires modifying the file
42// rsb-ts1-modified.c. Set the constant 'trials' on line 42 to the desired
43// number of throws. Then set kNumThrows below to the same number, and rebuild
44// OpenSpiel by running the script build_and_run_tests.sh.
45
46// Note that in his discussion of the results of the first competition, Darse
47// Billings observed that match length was not particularly important: "The
48// results were remarkably robust, and increasing the match length to 10000
49// turns or decreasing it to 400 turns had a negligible effect."
50// https://webdocs.cs.ualberta.ca/~darse/rsb-results1.html
