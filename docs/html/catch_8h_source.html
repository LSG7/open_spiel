24 // Catch is a single player game, often used for unit testing RL algorithms.
25 //
26 // The player must move a paddle to intercept a falling ball. The initial
27 // column of the ball is decided by chance. Each turn, the ball moves downwards
28 // while remaining in the initial column.
29 //
30 // Please note: In each turn, all actions (left, stay, right) are legal. This
31 // is different to the Python implementation of the game.
32 //
33 // References:
34 // a) Recurrent models of visual attention, 2014, Minh et al.
35 // (Advances in Neural Information Processing Systems 27, pages 2204â€“2212.)
36 // b) Behaviour Suite for Reinforcement Learning, 2019, Osband et al.
37 // (https://arxiv.org/abs/1908.03568)
38 //
39 // Parameters:
40 // "rows" int rows of the board (default = 10)
41 // "columns" int columns of the board (default = 5)
