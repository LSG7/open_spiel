41 // specified at compile time. Changing it requires modifying the file
42 // rsb-ts1-modified.c. Set the constant 'trials' on line 42 to the desired
43 // number of throws. Then set kNumThrows below to the same number, and rebuild
44 // OpenSpiel by running the script build_and_run_tests.sh.
