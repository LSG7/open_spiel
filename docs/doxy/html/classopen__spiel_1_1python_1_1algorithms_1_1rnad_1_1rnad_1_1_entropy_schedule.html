Detailed Description

An increasing list of steps where the regularisation network is updated.

Example
  EntropySchedule([3, 5, 10], [2, 4, 1])
  =>   [0, 3, 6, 11, 16, 21, 26, 36]
        | 3 x2 |      5 x4     | 10 x1

93 # the explicit schedule, and then we'd need to just use the last step
94 # in the schedule and apply the logic of
95 # ((learner_step - last_step) % last_iteration) == 0)

97 # The schedule might look like this:
98 # X----X-------X--X--X--X--------X
99 # learner_step | might be here ^ |
100 # or there ^ |
101 # or even past the schedule ^
102
103 # We need to deal with two cases below.
104 # Instead of going for the complicated conditional, let's just
105 # compute both and then do the A * s + B * (1 - s) with s being a bool
106 # selector between A and B.

108 # 1. assume learner_step is past the schedule,
109 # ie schedule[-1] <= learner_step.
110 last_size = self.schedule[-1] - self.schedule[-2]
111 last_start = self.schedule[-1] + (
112 learner_step - self.schedule[-1]) // last_size * last_size
113 # 2. assume learner_step is within the schedule.
114 start = jnp.amax(self.schedule * (self.schedule <= learner_step))
115 finish = jnp.amin(
116 self.schedule * (learner_step < self.schedule),
117 initial=self.schedule[-1],

119 size = finish - start

128 jnp.sum(learner_step == iteration_start + iteration_size - 1),
