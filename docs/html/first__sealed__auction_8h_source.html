26 // First-Price Sealed-Bid Auction:
27 // https://en.wikipedia.org/wiki/First-price_sealed-bid_auction
28 //
29 // Each player has a valuation of the target object from 1 to K, according to a
30 // uniform distribution, and places bids from 0 to (valuation - 1). The highest
31 // bidder gets reward (valuation - bid); the others get 0. In the case of a
32 // tie, the winner is randomly determined amongst the highest bidders.
33 //
34 // Parameters:
35 // "max_value" int maximum valuation (default = 10)
36 // "players" int number of players (default = 2)
