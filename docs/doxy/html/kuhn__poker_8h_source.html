27// A simple game that includes chance and imperfect information
28// http://en.wikipedia.org/wiki/Kuhn_poker
29//
30// For more information on this game (e.g. equilibrium sets, etc.) see
31// http://poker.cs.ualberta.ca/publications/AAAI05.pdf
32//
33// The multiplayer (n>2) version is the one described in
34// http://mlanctot.info/files/papers/aamas14sfrd-cfr-kuhn.pdf
35//
36// Parameters:
37// "players" int number of players (default = 2)
