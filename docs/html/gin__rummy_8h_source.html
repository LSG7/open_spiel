18 // Implementation of the classic card game:
19 // https://en.wikipedia.org/wiki/Gin_rummy
20 //
21 // Gin rummy is played with many variations. Here we closely follow
22 // the rules described in http://ginrummytournaments.com/pdfs/GRA_Rules.pdf
23 //
24 // A game consists of a single hand of gin (i.e. this implementation does not
25 // support a full game, consisting of multiple hands, played to some given
26 // point total, usually in the 100-300 point range).
27 //
28 // Gin is a large game with over 10^85 information states and a large number
29 // of states per information state. Off the deal there are 41 choose 10 =
30 // 1,121,099,408 possible opponent hands, compared to 50 choose 2 = 1,225 in
31 // heads up Texas hold 'em poker.
32 //
33 // Parameters:
34 // "oklahoma" bool use oklahoma variation? (default = false)
35 // "knock_card" int set a specific knock card (default = 10)
36 // "gin_bonus" int bonus for getting gin (default = 25)
37 // "undercut_bonus" int bonus for an undercut (default = 25)
38 // "num_ranks" int num ranks in deck (default = 13)
39 // "num_suits" int num suits in deck (default = 4)
40 // "hand_size" int num cards in player hand (default = 10)
