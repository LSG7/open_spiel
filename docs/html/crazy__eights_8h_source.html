18 // The game of crazy eights.
19 // See https://en.wikipedia.org/wiki/Crazy_Eights
20 // For 2~5 players, the game uses a standard 52-card deck.
21 // For >5 players, it uses 2 decks.
22 // Initially a player is randomly selected as the dealer.
23 // Then each player is dealt 5 cards (7 cards if there are 2 players).
24 // Then the dealer draws one card from the deck and turns it face up.
25 // Then started with the player on the dealer's right,
26 // the game goes counterclockwise
27 // by default (with an exception, details later).
28 // In each player's turn, it needs to play a card that either match the suit
29 // or the rank of the card on the top of the discard pile.
30 // And then place this card on the discard pile top for the next player to
31 // match. A player can play an 8 as a wild card, however, at anytime. If it does
32 // so then a color needs to be nominated for the next player to match. A player
33 // can also decide to draw cards from the dealer deck. Notice that it is the
34 // only action available if it does not have a available card to play at its
35 // turn. But it doesn't prevent the player to draw cards even if it has playable
36 // cards. However, the maximum number of cards a player can draw at its turn is
37 // bounded. If a player plays a card, it cannot draw at the current turn
38 // anymore. The game ends if a player has played all of its card. The other
39 // players are penalized according to the cards on their hand. That is, -50 for
40 // each 8, -10 for each court card, and -{face value} for others.
41 //
42 //
43 // The game can also incorporate other "special cards".
44 // These including:
45 // Skip: if a player plays Q, then the next player is skipped
46 // Reverse: if a player plays A, then the direction of play is reversed.
47 // Draw 2: if a player plays 2, then the next player should draw 2 cards.
48 // However, it admits stacking. That is, if the next player has 2, it can play
49 // it. And then the next player after it should draw 4 cards unless it plays
50 // draw 2 as well, etc. If a player starts to draw in this case, it must draw
51 // all the cards and then passes. I.e., if it draws a draw 2 card during the
52 // drawing, it is not allowed to play it.
53 //
54 // If the first card turned face up by the dealer is a special card,
55 // then it acts as if the dealer plays the card.
56 //
57 // If reshuffle = true, then the discard pile got reshuffle and become the new
58 // dealer card once exhausted.
59 //
60 // The action space of this game is as follows.
61 // action id 0, 1,..., 51: play/deal a card from the standard 52-card deck.
62 // action id 52: a player draw a card from the dealer's deck.
63 // action id 53: a player passes if it had already drawn max_draw_cards.
64 // action id 54, 55, 56, 57: a player nominate one of the four suit.
65 // (for chance) action id 0, 1,...., 51 are cards to be drawn
66 // action id 52, 53, ...., 52 + num_player-1: decide the dealer.
67 //
68 // An observation contains:
69 // (1) the current hand I have
70 // (2) the previous card and previous suit
71 // (3) starting from (my_idx + 1), the numbers of cards others have
72 // (4) whether currently it goes counterclockwise or not

185  return kNominateSuitActionBase + kNumSuits;
186  }
187  int MaxChanceOutcomes() const override {
188  return kDecideDealerActionBase + num_players_;
189  }
190  std::unique_ptr<State> NewInitialState() const override {
191  return absl::make_unique<CrazyEightsState>(shared_from_this(), num_players_,
192  max_draw_cards_,
193  use_special_cards_, reshuffle_);
194  }
195  int NumPlayers() const override { return num_players_; }
196  double MinUtility() const override {
197  return -kMaxPenality * (num_players_ > 5 ? 2 : 1);
198  }
199  double MaxUtility() const override { return 0.0; }
200  std::vector<int> ObservationTensorShape() const override {
201  int num_decks = num_players_ > 5 ? 2 : 1;
202  int base_observation_size =
203  (num_decks + 1) * kNumCards + kNumCards + kNumSuits +
204  (num_decks * kNumCards + 1) * (num_players_ - 1);
205  if (!use_special_cards_) {
206  return {base_observation_size};
207  } else {
208  return {base_observation_size + 1};
209  }
210  }
211  // In principle, the game can run indefinitely
212  int MaxGameLength() const override { return kMaxTurnLimit; }
213  int GetMaxDrawCards() const { return max_draw_cards_; }
214 
215  private:
216  int num_players_;
217  int max_draw_cards_;
218  bool use_special_cards_;
219  bool reshuffle_;
