24// Implementation of 'Deep Sea' exploration environment.
25//
26// This environment is designed as a stylized version of the 'exploration
27// chain', such as the classical river swim environment (Strehl & Littman '04,
28// https://ieeexplore.ieee.org/document/1374179). The observation is an N x N
29// grid, with a falling block starting in top left. Each timestep the agent can
30// move 'left' or 'right', which are mapped to discrete actions 0 and 1 on a
31// state-dependent level. There is a large reward of +1 in the bottom right
32// state, but this can be hard for many exploration algorithms to find.
33//
34// For more information, see papers:
35// [1] https://arxiv.org/abs/1703.07608
36// [2] https://arxiv.org/abs/1806.03335
37//
38// Parameters:
39// "size" int rows and columns (default = 5)
40// "seed" int seed for randomizing actions (default = 42)
41// "unscaled_move_cost" double move cost (default = 0.01)
42// "randomize_actions" bool state dependent actions (default = true)
43
