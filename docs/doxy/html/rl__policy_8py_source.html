29 """Transforms an RL Agent into an OpenSpiel policy.
30
31 Args:
32 rl_class: An OpenSpiel class inheriting from 'rl_agent.AbstractAgent' such
33 as policy_gradient.PolicyGradient or dqn.DQN.
34
35 Returns:
36 An RLPolicy class that wraps around an instance of rl_class to transform it
37 into a policy.
38 """

98 def step(self, time_step, is_evaluation=False):
99 # The _frozen attribute freezes the weights of the current policy. This
100 # effect is achieved by considering that we always are evaluating when the

106 def freeze(self):
107 """This method freezes the policy's weights.
108
109 The weight freezing effect is implemented by preventing any training to
110 take place through calls to the step function. The weights are therefore
111 not effectively frozen, and unconventional calls may trigger weights
112 training.
113
114 The weight-freezing effect is especially needed in PSRO, where all
115 policies that aren't being trained by the oracle must be static. Freezing
116 trained policies permitted us not to change how 'step' was called when
117 introducing self-play (By not changing 'is_evaluation' depending on the
118 current player).
