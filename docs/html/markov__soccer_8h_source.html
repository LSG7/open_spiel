26 // This is the soccer game from the MinimaxQ paper. See
27 // "Markov Games as a Framework for Reinforcement Learning", Littman '94.
28 // http://www.cs.duke.edu/courses/spring07/cps296.3/littman94markov.pdf
29 //
30 // Parameters:
31 // "horizon" int max number of moves before draw (default = 1000)
32 // "grid" string String representation of grid.
33 // Empty spaces are '.', possible ball starting
34 // locations are 'O' and player A and B starting
35 // points are 'A' and 'B' respectively.
