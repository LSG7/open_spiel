25// Lewis Signaling Game: https://en.wikipedia.org/wiki/Lewis_signaling_game
26//
27// First agent (sender) receives a random private state from a set of N states.
28// It then sends a message from a set of M messages to the second agent
29// (receiver). Finally, the receiver takes an action after observing the
30// message. An N*N payoff matrix is used to calculate the reward based on the
31// state received by the sender and the action taken by the receiver. Both
32// agents receive the same reward.
33//
34// Parameters:
35// "num_states" int number of distinct states (N) (default = 3)
36// "num_messages" int number of distinct messages (M) (default = 3)
37// "payoffs" string string with comma separated payoff values
38// (N*N elements required)
39// (default = flattened identity matrix)
40
