18 // The game of dou dizhu (the three-player version).
19 // For a general description of the rules, see
20 // https://en.wikipedia.org/wiki/Dou_dizhu It uses a standard 54-card deck
21 // (including two Jokers). The game starts by randomly picking one card face up,
22 // which is then inserted into the shuffled deck. Then each player is dealt 17
23 // cards. Then the bidding phase starts. The player who got the face-up card
24 // becomes the first one to bid. Bidding round ends if (1) no one bids (2) two
25 // consecutive passes (3) maximum bid 3 was bidded. The one who wins the bidding
26 // phase becomes dizhu (landlord). Dizhu get the remaining 3 cards. The other
27 // players are called peasants. Starting with dizhu, the playing phase
28 // consisting of multiple tricks. The leader of a trick can play several
29 // allowable categories of hands. The players during a trick can only pass or
30 // play hands of the same pattern of higher rank. A player becomes the winner of
31 // a trick if the other two players passes. And then it becomes the leader of
32 // the next trick. In this game, suits DO NOT MATTER.
33 //
34 // The allowable categories of hands:
35 // Solo: a single card
36 // SoloChain: >=5 consecutive cards in rank, e.g., 34567
37 // Pair: a pair of card with the same rank
38 // PairChain: >= 3 consecutive pairs. e.g., 334455
39 // Trio: three of a rank. e.g., 444
40 // TrioWithSolo: a trio + a single hand. e.g., 3334
41 // Trio With Pair: a trio + a pair. e.g., 33344
42 // Airplane (TrioChain). >=2 consecutive trio. e.g., 333-444
43 // Airplane+solo. airplane where each trio carries a solo. e.g., 333-444-5-6
44 // Airplane+pair. airplane where each trio carries a pair. e.g., 333-444-55-66
45 // Bomb. Four of a rank. e.g., 4444
46 // Rocket. Two jokers
47 //
48 // Some other rules:
49 // The order for solo card is: ColoredJoker>BlackWhiteJoker>2>A>K>Q>....>4>3
50 // For combination hands, the primal part determines the order.
51 // e.g. the primal part of 333-444-5-6 is 333-444
52 // 2s and Jokers cannot be in a chain.
53 // Rocket dominates all other hands.
54 // A bomb dominates all other hands except rocket or bombs of higher rank.
55 // Bomb/rocket cannot appear in an airplane combination
56 // E.g., 333-444-555-666-7777 is prohibited.
57 // But in this implementation any pair and any trio can be kickers
58 // For more, see https://rezunli96.github.io/blog/doudizhu_count.html
59 //
60 // A game ends if a player has played all their cards.
61 // The winning bid determines the initial stake.
62 // Each bomb played doubles the stake.
63 // And if (1) both peasants do not play any cards
64 // (2) dizhu does not play any cards after its first hand, then it's called
65 // spring. And the stake is also doubled.
