111 // Returns a map of infostate strings to a vector of history nodes with
112 // corresponding counter-factual probabilities, where counter-factual
113 // probabilities are calculatd using the passed policy for the opponent's
114 // actions, a probability of 1 for all of the best_responder's actions, and the
115 // natural chance probabilty for all change actions. We return all infosets
116 // (i.e. all sets of history nodes grouped by infostate) for the sub-game rooted
117 // at state, from the perspective of the player with id best_responder.
118 absl::flat_hash_map<std::string, std::vector<std::pair<HistoryNode*, double>>>
119 GetAllInfoSets(std::unique_ptr<State> state, Player best_responder,
