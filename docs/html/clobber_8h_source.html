18 // Implementation of the board game Clobber.
19 // https://en.wikipedia.org/wiki/Clobber
20 //
21 // Some notes about this implementation:
22 // - The two players:
23 // Clobber is a two player game. The two players in this
24 // implementation are 'o' (White, 0) and 'x' (Black, 1). In the
25 // default board of any size, the bottom left corner is always
26 // 'o' and continues in a checkerboard pattern from there. 'o'
27 // moves first in the default board.
28 // - Custom boards:
29 // A custom board can be used to initialize a state when calling
30 // either the ClobberState(rows, columns, board_string) constructer
31 // or ClobberGame's method NewInitialString(board_string). Where
32 // 'rows' and 'columns' are the number of rows and columns on the
33 // board respectively, and 'board_string' is a string representing
34 // the board. The format of board string is as follows:
35 // - The first character is either a '0' or '1', this indicates
36 // which player's turn it is (white or black respectively).
37 // - The next characters are either 'o', 'x', or '.' which
38 // represent white pieces, black pieces, or empty cells
39 // respectively. There must be rows * columns number of these
40 // characters following the first character.
41 // For example, a state initialized from "1x.o.xo.x." on a game with
42 // 3 rows and 3 columns would have 'x' (Black, 1) play first on a
43 // 3x3 board with configuration:
44 // x.o
45 // .xo
46 // .x.
47 // - Observation tensor:
48 // This version implements a 3-plane observation tensor. Each plane
49 // has equal dimensions as the board. The first plane contains 1's\
50 // where the current player's pieces are, and 0's elsewhere. The
51 // next plane contains 1's where their opponent's pieces are, and
52 // 0's elsewhere. Finally, the last plane consists of 1's where the
53 // empty cells are, and 0's elsewhere.

149  // On every turn, one piece is taken out. The longest game occurs
150  // when the last player takes out the only remaining opponenent's
151  // piece with their last piece. Therefore, there is still one piece on
152  // the board. Hence, the maximum number of moves is # of cells - 1.
153  int MaxGameLength() const override { return rows_ * columns_ - 1; }
