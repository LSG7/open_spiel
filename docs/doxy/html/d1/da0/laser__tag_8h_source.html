27// A fully observable version of the first-person gridworld laser tag game from
28// [1,2] with identical mechanics. This implementation includes support for
29// both fully observable (the default) and first-person partially observable
30// modes. The current grid is "small2" from [2].
31//
32// When run in partially observable mode each agent has a local field-of-view,
33// and sees (by default) 17 spaces in front, 10 to either side, and 2 spaces
34// behind (per [2]). Each agent's observation is encoded as a 4x20x21 tensor,
35// where each plane encodes the presence of player A, player B, empty, obstacle,
36// respectively (same as fully observable tensor observations). The dimensions
37// - front, side, and back - of the field of vision can be changed (see
38// parameters below).
39//
40// [1] Leibo et al. Multi-agent Reinforcement Learning in Sequential Social
41// Dilemmas. https://arxiv.org/abs/1702.03037
42// [2] Lanctot et al. A Unified Game-Theoretic Approach to Multiagent
43// Reinforcement Learning", https://arxiv.org/abs/1711.00832
44//
45// Parameters:
46// "horizon" int Number of steps per episode. If this is < 0, then
47// the episode ends after the first tag.
48// (default = 1000).
49// "zero_sum" bool If set, rewards are +1 for a tag and -1 for being
50// tagged. Otherwise, there there is only positive
51// reward of +1 per tag. (default = false).
52// "grid" string String representation of grid.
53// Empty spaces are '.', obstacles are '*', spawn
54// points are 'S' (there must be four of these).
55// "fully_obs" bool If set, the environment is full observable.
56// Otherwise, the environment is partially
57// observable (default = true)
58// "obs_front" int Number of squares each agent sees in front of
59// themself (only used if fully_obs=false)
60// (default=17)
61// "obs_back" int Number of squares each agent sees behind themself
62// (only used if fully_obs=false) (default=2)
63// "obs_side" int Number of squares each agent sees to either side
64// of themself (only used if fully_obs=false)
65// (default=10)
