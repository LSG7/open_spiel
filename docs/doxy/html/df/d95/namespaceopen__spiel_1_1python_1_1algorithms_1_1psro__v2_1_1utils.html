Returns element-wise rounded version y of a vector x, with sum(x)==sum(y).

E.g., if x = array([3.37625333, 2.27920304, 4.34454364]), note sum(x) == 10.

however, the rounded counterpart y will have sum(y) == 10.

  x: a vector.

143def round_maintain_sum(x):
144 """Returns element-wise rounded version y of a vector x, with sum(x)==sum(y).
145
146 E.g., if x = array([3.37625333, 2.27920304, 4.34454364]), note sum(x) == 10.
147 However, naively doing y = np.round(x) yields sum(y) == 9. In this function,
148 however, the rounded counterpart y will have sum(y) == 10.
149
150 Args:
151 x: a vector.
152 """
153 y = np.floor(x)
154 sum_diff = round(sum(x)) - sum(y) # Difference of original vs. floored sum

156 y[indices] += 1 # Add the missing mass to the elements with the most missing
157 return y
