16 """Representation of a value for a game.
17 
18 This is a standard representation for passing value functions into algorithms,
19 with currently the following implementations:
20 
21 The main way of using a value is to call `value(state)`
22 or `value(state, action)`.
23 
24 We will prevent calling a value on a state action on a MEAN_FIELD state.
25 
26 The state can be a pyspiel.State object or its string representation. For a
27 particular ValueFunction instance, you should use only one or the other. The
28 behavior may be undefined for mixed usage depending on the implementation.
