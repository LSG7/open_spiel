27// An implementation of Mensch-Aergere-Dich-Nicht (see
28// https://en.wikipedia.org/wiki/Mensch_%C3%A4rgere_Dich_nicht)
29//
30// Rules used:
31// - start field must be cleared as soon as possible
32// - throwing out own pieces is not possible
33// - only one dice roll even if no move is possible except if dice roll was
34// a six, in this case, same player may roll again
35// - pieces may jump over each other on four final fields
36//
37// Parameters:
38// - players: Number of Players (2 to 4)
39// - twoPlayersOpposite:
40// If two players play, two different settings are possible:
41// Either players can play side by side or they can play on opposite sides.
42// Since opposite sides are more fair, default value is true.
43

66// Action modelling (with ideas from Marc Lancot):
67// The first action [0] is to pass (necessary if player cannot move any
68// piece). The second action is to bring in a new piece. Once a piece is
69// on the field, there are 43 fields a piece can stand on and be moved away
70// from that field. Actions are coded as the field a move starts from, from
71// each player's own PoV. That means that action 2 means to move a piece on
72// field 0 for player 0 but a piece on field 10 for player 1 and so on. So
73// there are 43 actions for moves, one action to bring in a new piece and
74// one action to pass. Total number of possible actions is 45
75// ({ 0, 1, 2, ..., 44 }).
76inline constexpr const int kNumDistinctActions = 45;

79inline constexpr const Action kBringInAction = 1;
80inline constexpr const Action kFieldActionsOffset = 2;

295 std::vector<int> ObservationTensorShape() const override {
296 // Encode each field on the board as four doubles:
297 // - One double for whether there is a piece of player 1 (1 or 0).
298 // - One double for whether there is a piece of player 2 (1 or 0).
299 // - One double for whether there is a piece of player 3 (1 or 0).
300 // - One double for whether there is a piece of player 4 (1 or 0).
301 // (effectively that is one-hot encoded player number)
302 //
303 // Return a vector encoding:
304 // - Every field.
305 // - One double for the number of pieces outside the board for player 1.
306 // - One double for the number of pieces outside the board for player 2.
307 // - One double for the number of pieces outside the board for player 3.
308 // - One double for the number of pieces outside the board for player 4.
309 // - One double for whether it's player 1's turn (1 or 0).
310 // - One double for whether it's player 2's turn (1 or 0).
311 // - One double for whether it's player 3's turn (1 or 0).
312 // - One double for whether it's player 4's turn (1 or 0).
313 // (If it's chance player's turn, all four doubles are 0.)
314 // - One double for whether dice roll is a 1 (1 or 0).
315 // - One double for whether dice roll is a 2 (1 or 0).
316 // - One double for whether dice roll is a 3 (1 or 0).
317 // - One double for whether dice roll is a 4 (1 or 0).
318 // - One double for whether dice roll is a 5 (1 or 0).
319 // - One double for whether dice roll is a 6 (1 or 0).
320 // (If it's chance player's turn, all six doubles are 0.)
321
322 return {kStateEncodingSize};
323 }
324
325 private:
326 bool two_player_opposite_;
327 int num_players_;
328};
